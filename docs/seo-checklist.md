# Local SEO Checklist

## Google Business Profile
- Complete all business details
- Select correct primary category
- Add services and descriptions
- Upload real images
- Post updates regularly

## Website Optimization
- Use location-specific keywords
- Create service-specific pages
- Optimize title tags and headings

## Reviews
- Ask every satisfied customer
- Respond to all reviews
This checklist is part of a broader SEO system used in real campaigns.
